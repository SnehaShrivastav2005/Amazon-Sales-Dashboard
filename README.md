#AMAZON SALES DASHBOARD | POWER BI

## Project Overview
This project is an interactive **AMAZON SALES DASHBOARD** built using **POWER BI**.
It provides insights into sales performance, trends, and key business matrices to support data - driven decision-making.

## Tools and Technologies
-Power BI
-Microsoft Excel

## Key Insights
-Sales performance by category
-Regional sales distribution
-Monthly nad yearly sales trends

## NOTE: 
Github cannot preview large Power BI (.pbix) files.
Please download the file and open it using **POWER BI DESKTOP** to view the dashboard.
